Setup Guide



Apple iOS

  1. 1. Установите V2Box из App Store.
  2. 2. Скопируйте ключ на главной вкладке этого бота.
  3. 3. В приложении нажмите "+" вверху → "Import from Clipboard".
  4. 4. Нажмите "Connect" и разрешите добавление VPN-конфигурации.
TV

Android & Android TV

  1. 1. Установите v2rayNG (есть в Play Store).
  2. 2. Нажмите "+" → "Импорт из буфера обмена".
  3. 3. Выберите сервер в списке и нажмите круглую кнопку "V" снизу.
  4. • Для TV: используйте флешку для установки .apk или TV-браузер.
⌘

Windows / Mac

Используйте Nekoray (Windows/Mac). Просто нажмите Ctrl+V внутри программы после копирования ключа. Затем правой кнопкой → "Start".
